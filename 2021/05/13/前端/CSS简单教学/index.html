基础选择器

用来选择页面上的元素,注意style中写css的代码。

1、标签选择器（标签{}）

2、类选择器（.class{}）

3、id选择器（#id名{}）

id命名全局唯一

4、优先级

与顺序与关，id选择器>class选择器>标签选择器

层次选择器

1、后代选择器

（ul）元素后的所有（p）元素实现

2、子选择器

选择作为（body）元素子元素的（p）元素实现

3、相邻兄弟选择器

选择（.a）紧凑的后一（p）元素实现，要有相同父类

4、通用选择器

选择（.a）后所有的（p）某种元素实现

属性选择器

  • 属性判断，可用正则表达式
    • = 绝对等于
    • *= 包含这个元素
    • ^= 以这个开头
    • $= 以这个结尾

美化网页

  • span标签，展现想突出的字

1、字体样式（font）

  • font-style 字体样式
  • font-variant 字体异体（小型大写字母）
  • font-weight 字体粗细
  • font-size/line-height 字体尺寸/行高
  • font-family 字体体系

2、文本样式

3、超链接伪类（主要是hover）

hover是鼠标悬浮于内容上时的对应实现。

4、列表

  • list-style
    • none 去标点
    • circle 空心圆
    • decimal 数字
    • square 正方形

5、背景

  • image
    • background-image: url(“···”) 图片填满范围，要定义长宽
    • background-repeat: repeat-x 填满横轴
    • background-repeat: repeat-y 填满纵轴
    • background-repeat: no-repeat 只填一张图

  • 渐变色

    linear-gradient(to right, red,orange,yellow,green,blue,indigo,violet)

盒子模型

注意首先要写一个框架出来

  • margin：外边距
  • border：边框
  • padding：内边距

1、边框（border）

  • border-width：边框宽度
  • border-style：边框样式
  • border-color：边框颜色

2、内外边距（padding、margin）

  • 属性为上右下左顺时针的4个元素，若有缺省，则按bottom=top、left=right赋值

  • 居中元素（要有块元素）

3、圆角边框

  • border-radius

    参数顺序左上、右上、右下、左下，有缺省时按对角线赋值，可以画图形。

1、display

  • block：块元素
  • inline：行内元素
  • inline-block：块元素，但可以在一行
  • none：隐藏
